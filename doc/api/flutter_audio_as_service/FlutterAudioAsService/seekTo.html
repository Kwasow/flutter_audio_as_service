Future<void> seekTo (Duration seekTo)

Seeks to a specified positions

Implementation

static Future<void> seekTo(Duration seekTo) async {
  await _checkIfBound();
  await _nativeChannel.invokeMethod("seekTo", {
    "seekToInMs": seekTo.inMilliseconds,
  });
}
